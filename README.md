# Customer Revenue Intelligence System

## 📊 Project Overview
This project analyzes customer revenue patterns to identify high-value customers, contribution percentage, and business insights.

## 🛠️ Tools Used
- SQL Server
- Power BI
- Excel

## 📂 Dataset
Contains sales data including customer, region, product, and amount.

## 🔍 Key Analysis
- Customer segmentation (High / Medium / Low)
- Contribution % analysis
- Top customers identification
- Revenue trends

## 📊 Dashboard
Power BI dashboard showing:
- Total Revenue
- Customer Contribution
- Trends over time

## 💡 Insights
- Top 20% customers contribute majority revenue
- Regional performance differences observed
- Repeat customers generate higher revenue

## 🚀 Outcome
This project helps businesses improve customer retention and revenue strategies.
